OUR TOOL

ABOUT

Provide a URL to find products that are similar to any clothing items found in the image. Obtain full product names, IDs, prices and discounts, and direct links to the corresponding brand's web store.

MAIN GOAL

With this API, you can:

  • Provide an image of a product you have interest in via a URL, and get similar products currently available in Zara.
  • Get product IDs and names, the current price and also the original price if there's a discount, and a link to the product's web store, and the brand behind the product.
  • Paginate through results to easily manage large data sets.

CONTEXT

This API is available through the Inditex Developer Portal. To be able to use it, you first need to sign up, create an application, and then add this API to it.
